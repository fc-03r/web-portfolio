# Web Portfolio

## 概要
本リポジトリは、職業訓練校での学習および自主制作として作成したWebサイトのポートフォリオです。

猫カフェをテーマにした架空サイトに加え、ランダムカラーパレット生成、ゲームフォントプレビュー、自動文字送りアニメーションなどのUI表現を実装しています。

※本作品はPHPおよびデータベースを用いた動的Webアプリケーションのため、GitHub上では動作せず、実行にはサーバー環境が必要です。  
本リポジトリではソースコードおよび設計内容を公開しています。

## 使用技術
- HTML / CSS
- JavaScript
- PHP
- GitHubでのバージョン管理

## 工夫した点
- セクションごとにCSSを分割し、保守性を意識した設計
- 視認性とユーザー導線を意識したレイアウト構築
- アニメーションやUI表現による視覚的な分かりやすさの向上
